S.No. Header File Description
1.
  • Header File (Github) :
    - Bitmap_Font_Manager.hpp
  • Header in Use (Demo Code) :
    - bitmap_rendering_demo1.html
  • Single Header Class file, class named - Bitmap_Font
  • Necessary Libraries - SDL_Image, SDL, cstring, cmath, iostream
  • Methods -
    1. Bitmap_Font () - constructor
    2. ~Bitmap_Font () - destructor
    3. void free_image () - frees loaded texture
    4. void load_image ( SDL_Renderer* i_renderer, std::string path ) - loads the bitmap font image
    5. void update_format ( std::string i_input ) - updates the format like 16x6 or 16x16 (full ASCII table)
    6. void set_font_size ( int i_font_size ) - sets the font size, should be corresponding to the width and height of characters
    7. int get_row ( char i_value ) - returns index row of that character ( no need to call )
    8. int get_col ( char i_value ) - returns index col of that character ( no need to call )
    9. void render_text ( SDL_Renderer* i_renderer, int x, int y, std::string i_text ) - renders the the text inputted in i_text
    10. void render_text_advanced ( SDL_Renderer* i_renderer, int x, int y, std::string i_text, bool i_animation ) - same as render_text but can format outputted text with characters like \n and \r
    11. void change_color ( int r, int g, int b ) - changes the color of the text (if the bitmap is in 'grey' or 'white' color results the best)
  • Members -
    1. SDL_Texture* l_texture - for bitmap image texture
    2. int width - width of the image texture
    3. int height - height of the image texture
    4. bool is_full - format of bitmap image (16x16 / 16x6)
    5. int row - size of row, it changes depending of format - is_full
    6. int col - size of col, it does not change as per now
    7. int font_size - size of the font, remember if size of a character in your bitmap is 24 this should be 24
    8. int col_switch - for the rendering_advanced text, switch for col ( no need to call )
    9. int row_switch - for the rendering_advanced text, switch for row ( no need to call )
2.
  • Header File (Github) :
    - WindowManager_class.hpp
  • Header in Use (Demo Code) :
    - using_window_manager.html
  • Single Header class File, class Named - WindowManager
  • Necessary Libraries - iostream, SDL
  • Methods -
    1. ~WindowManager () - Destructor
    2. void free_window_manager () - destroys renderer and window
    3. bool create_window_renderer () - creates window and renderer
    4. void set_width ( int i_width ) - sets the window's width, should be called before creating window and renderer
    5. void set_height ( int i_height ) - sets the window's height, should be called before creating window and renderer
    6. void set_window_title ( std::string i_window_title ) - sets the window title, should be called before creating window and renderer
  • Members -
    1. SDL_Window* window - window ptr
    2. SDL_Renderer* renderer - renderer ptr
    3. std::string window_title - for window's title (default - "window-1")
    4. int width - for window's width (default: 640)
    5. int height - for window's height (default: 480)
  • It is very simple, currently you cannot change the renderer type and window type, but in future it will be available
